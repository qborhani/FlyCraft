FlyCraft v1.4 Documentation
==========================

1. Core Features
---------------
- Toggle flight mode for players
- Configurable flight duration
- Configurable cooldown system
- Customizable fly speeds (1-10)
- Permission-based access
- Welcome messages for new players
- Full Discord integration

2. Commands
----------
2.1 In-Game Commands:
    /fly [player]
    - Description: Toggle flying mode
    - Permission: flycraft.use
    - Aliases: flymode
    - Optional: [player] - Toggle flight for another player (requires flycraft.others)

    /flyspeed <1-10>
    - Description: Change flying speed
    - Permission: flycraft.use
    - Range: 1 (slowest) to 10 (fastest)

    /flycraftreload
    - Description: Reloads the plugin configuration
    - Permission: flycraft.reload
    - Aliases: fcr, flyreload

2.2 Discord Commands:
    !startfly <player>
    - Enables flight for specified player
    - Requires configured Discord role

    !stopfly <player>
    - Disables flight for specified player
    - Requires configured Discord role

    !flyspeed <player> <1-10>
    - Sets player's flight speed
    - Requires configured Discord role

3. Permissions
-------------
flycraft.use
- Description: Allows using flight mode
- Default: true

flycraft.reload
- Description: Allows reloading configuration
- Default: op

flycraft.others
- Description: Allows toggling flight for others
- Default: op

4. Configuration (config.yml)
---------------------------
4.1 Flight Settings:
- flight-duration: Duration of flight in seconds (default: 300)
- flight-cooldown: Cooldown between flights in seconds (default: 60)
- welcome-message-enabled: Show welcome message to new players

4.2 Discord Integration:
- Enabled/Disabled toggle
- Channel ID configuration
- Bot token configuration
- Custom bot status settings
- Role-based command permissions
- Customizable message formats
- Embed support with custom colors

4.3 Bot Settings:
- Status options: ONLINE, IDLE, DO_NOT_DISTURB, INVISIBLE
- Activity types: PLAYING, WATCHING, LISTENING, COMPETING
- Custom activity message with placeholders:
  %online_players%
  %max_players%
  %server_name%
